# Mental-Health
# Mental Health Stress Level Predictor (Bhutan Context)

This project provides a machine-learning based Streamlit application that predicts mental-health stress levels using survey-like features relevant to Bhutan.

## Features
- Synthetic dataset generator
- Random Forest classifier
- Streamlit interface for predictions
- Easy to expand with real survey data

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Train the ML model
python train_model.py

### 3. Run the Streamlit application
streamlit run app.py
